One-line: the primary call-to-action button — five Material 3 emphasis levels.

```jsx
<Button variant="filled" icon="bolt">AX 도입 문의</Button>
<Button variant="tonal">더 알아보기</Button>
<Button variant="outlined" trailingIcon="arrow_forward">자세히</Button>
<Button variant="text">취소</Button>
```

Variants by emphasis (high→low): filled, tonal, elevated, outlined, text. Pass `href` to render an `<a>`. Add `icon` / `trailingIcon` (Material Symbols names).
