One-line: the Material 3 icon glyph — wraps Material Symbols Rounded; use anywhere an icon is needed.

```jsx
<Icon name="search" />
<Icon name="bolt" fill size={20} />
<Icon name="arrow_forward" weight={500} />
```

Icon names are Material Symbols ligatures (search, home, settings, arrow_forward, bolt, auto_awesome, …). `fill` toggles the filled variant; `weight` (100–700) and `size` tune emphasis.
